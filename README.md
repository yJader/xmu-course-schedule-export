# 厦门大学课程表导出助手-ICS (功能修复版)

本脚本是基于 [yangqian](https://greasyfork.org/zh-CN/users/1392448) 的原创脚本 [厦门大学课程表导出助手-ICS](https://greasyfork.org/zh-CN/scripts/522668) 进行的维护和更新。

原脚本因厦门大学教务系统前端代码更新已失效，此版本旨在修复问题并增加新功能，感谢原作者的开创性工作！

特别感谢Gemini&Codex为本脚本的快速完成做出的贡献

---

### 主要更新日志 (v2025.09.15)

* **[核心]** 适配了 2025 秋季学期的新版教务系统页面结构。
* **[修复]** 重构了时间计算逻辑，能正确处理 **1节、2节、3节** 等任意时长的连续课程。
* **[修复]** 解决了因页面隐藏元素导致的课程被**重复导出**的严重 BUG。
* **[优化]** 将数据提取方式更新为更稳健的 `querySelector`，以应对未来页面微小调整。
* **[新增]** 内置 `DEBUG` (调试) 模式开关，方便未来排查问题。
* **[新增]** 课程提醒功能, 在日程中为课程添加提醒时间

### 使用说明

1. 安装脚本后，在厦门大学教务系统的“学生课程表”页面，顶部会出现一个【导出ICS】按钮。
2. 页面会新增“开始日期”输入框，可直接在界面中修改导出的起始日期，并按学期自动记住你的选择。
3. 若未手动设置，脚本会按学期自动给出默认值：
   * 春季学期：后一年 3 月第一个周一
   * 秋季学期：前一年 9 月第一个周一
   * 夏季学期：后一年 7 月第一个周一
4. 点击【恢复默认】可将当前学期日期重置为自动计算值。
5. 点击【导出ICS】即可下载 `.ics` 日历文件，可导入到手机或电脑的日历应用中。
